# DP (동적계획법)
- 메모리를 적절하게 사용해서 수행 시간 효율성을 향상시키는 방법
- 메모이제이션을 사용한다.
- 탑다운, 보텀업 방식으로 나뉜다.
- 조건 : 최적 부분 구조 (큰 문제를 작은 문제로 나누기), 중복 부분 문제 (동일한 작은 문제 반복)
